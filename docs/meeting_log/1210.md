## 1. react UI설계

- 반응형, 컴포넌트 변환(페이지 이동)시 전체 container박스의 높이가 변할 수 있는 문제.

  - 해결책 : scroll, pagination이용

- UI에 기능 삽입

  - 캐러셀 : 모든 전통주 중 random으로 3개 정보

    - 들어갈 정보 : 이미지 , 설명(일단 이미지로 가고 다음주에 한 번)

  - 첫 번째 칸 : 전통주 취향 test
  - 오른쪽 칸 : 전통주 장점 홍보 페이지
  - 왼쪽 가장 아래 칸 : 술 전체보기
  - 오른쪽 가장 아래 : 전통주점 찾기
  - navbar : main부분에 있는 기능들을 전부 navbar로 넣음
